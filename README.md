## 💡主要功能

1. 支持**全屏**截图
2. 支持**活动窗口**截图
3. 支持截取**光标**
4. 支持**热键**截图
5. 支持**自由框选截图**
6. 支持**录制GIF**
7. 支持**自动保存**图片
8. 支持**多显示屏**（仅在双显示屏下测试）
9. 支持**最小化**到**托盘**
10. 支持**跟随系统启动**
11. ......
